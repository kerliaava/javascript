This project is inspired by the 30 day vanilla Javascript challenge by Wesbos covering the very essentials of this programming language. In addition to Javascript, the project allows me to further deepen my knowledge in HTML5, CSS3, Sass and UI design. Throughout the projects I am using some inspiring artwork from my favourite painters.
